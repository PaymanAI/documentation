## Quick Start

1. Visit [Payman Dashboard](https://app.paymanai.com)
2. Sign up or sign in
3. Navigate to **Access Keys** section
4. Click `Create Access Key` Button
5. Copy your key immediately - it's only shown once

<Warning>
	Store your API key securely in environment variables. Never commit it to code.
</Warning>

## Environment Setup

```bash
# .env file
PAYMAN_API_SECRET=your_access_key_here
```


<Warning>
	USD/USDC Wallet transactions involve real money. Test thoroughly with Test Wallets first.
</Warning>
