## Environments

| Environment | Base URL |
|------------|----------|
| Sandbox | `https://agent-sandbox.payman.ai/api` |
| Production | `https://agent.payman.ai/api` |

## Authentication

All API requests require authentication using the `x-payman-api-secret` header. Get your API key from the [Payman Dashboard](https://app.paymanai.com).

```bash
x-payman-api-secret: sk_test_...
```

## Quick Start

<CodeGroup>

```bash curl
curl https://agent-sandbox.payman.ai/api/payments/send-payment \
  -H "x-payman-api-secret: sk_test_..." \
  -H "Accept: application/vnd.payman.v1+json" \
  -H "Content-Type: application/json" \
  -d '{
    "paymentDestinationId": "dest_123",
    "amountDecimal": 50.00,
    "memo": "Service payment"
  }'
```

```typescript node-fetch
const response = await fetch('https://agent-sandbox.payman.ai/api/payments/send-payment', {
  method: 'POST',
  headers: {
    'x-payman-api-secret': 'sk_test_...',
    'Accept': 'application/vnd.payman.v1+json',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    paymentDestinationId: 'dest_123',
    amountDecimal: 50.00,
    memo: 'Service payment'
  })
});

const data = await response.json();
```

```typescript node-axios
import axios from 'axios';

const response = await axios.post(
  'https://agent-sandbox.payman.ai/api/payments/send-payment',
  {
    paymentDestinationId: 'dest_123',
    amountDecimal: 50.00,
    memo: 'Service payment'
  },
  {
    headers: {
      'x-payman-api-secret': 'sk_test_...',
      'Accept': 'application/vnd.payman.v1+json',
      'Content-Type': 'application/json'
    }
  }
);

const data = response.data;
```

```python python-requests
import requests

response = requests.post(
    'https://agent-sandbox.payman.ai/api/payments/send-payment',
    headers={
        'x-payman-api-secret': 'sk_test_...',
        'Accept': 'application/vnd.payman.v1+json',
        'Content-Type': 'application/json'
    },
    json={
        'paymentDestinationId': 'dest_123',
        'amountDecimal': 50.00,
        'memo': 'Service payment'
    }
)

data = response.json()
```

</CodeGroup>

## Response Format

All responses use standard HTTP status codes and return JSON in the following format:

```json
{
  "status": 200,
  "data": {
    // Response data specific to each endpoint
  }
}
```
