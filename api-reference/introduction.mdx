## Overview

The Payman API enables AI Agents to send and receive payments securely. All API access is over HTTPS and endpoints are accessed from `https://agent.payman.ai/api`.

```bash
curl https://agent.payman.ai/api/payments/send-payment \
  -H "x-payman-api-secret: sk_test_..." \
  -d '{
    "paymentDestinationId": "dest_123",
    "amountDecimal": 50.00,
    "memo": "Service payment"
  }'
```

## Authentication

All API requests require the `x-payman-api-secret` header:

```bash
'x-payman-api-secret: sk_test_...'
```

Get your API key from the [Payman Dashboard](https://app.paymanai.com).

<Note>
  Keep your API key secure. Never share it in public repositories or client-side code.
</Note>

## Environments

```bash
# Production
https://agent.payman.ai/api

# Test Mode
https://agent-sandbox.payman.ai/api
```

Test mode provides a sandbox environment with simulated funds for development and testing.

## Response Format

All responses use standard HTTP status codes and return JSON:

```json
{
  "status": 200,
  "timestamp": "2024-01-22T10:00:00Z",
  "data": {
    "reference": "pay_123abc",
    "amount": 5000,
    "status": "pending"
  }
}
```

## Versioning

Include the API version in the Accept header:

```bash
Accept: application/vnd.payman.v1+json
```
