## Overview

Payman enables AI Agents to send and receive payments securely. Build AI applications that can handle real money with built-in safety controls and compliance.

```javascript
const payment = await payman.payments.sendPayment({
  paymentDestinationId: 'dest_123',
  amountDecimal: 50.00,
  memo: 'Service payment'
});
```

## Features

### Secure Payments
- ACH bank transfers
- Agent-to-agent instant transfers
- Crypto payments (Coming Soon)
- Customer deposits

### Built for AI
- AI-first API design
- Framework integrations (Langchain, CrewAI, Vercel AI + More)
- Automatic spend controls
- Real-time webhooks

### Enterprise Ready
- SOC 2 compliance
- Bank-grade security
- Complete audit trails
- Detailed reporting
