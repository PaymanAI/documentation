## The Financial Layer for AI Agents

Payman provides the critical financial infrastructure that lets AI Agents handle money safely and securely.
Build AI applications that can receive deposits, manage balances, and send payments - all with built-in compliance and controls.

🔒 Built On USD financial rails <br />
🛡️ Built-in spend limits and controls <br />
🔐 Separate financial accounts for AI Agents <br />
📊 Complete audit trails <br />
