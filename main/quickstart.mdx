## Overview

This guide will help you start sending payments with your AI Agent. By the end, you’ll be able to:

1. Get your API credentials
2. Create a payment destination
3. Send your first payment
4. Add AI capabilities

---

## 1. Prerequisites

Before you begin, ensure you have:

-   A **Payman AI** account. Sign up at [Payman AI Dashboard](https://app.paymanai.com)
-   A **development environment** (Node.js, Next.js)
-   An **AI provider** (OpenAI, Anthropic, etc.)

---

## 2. Get API Credentials

Follow these steps to obtain your API credentials:

1. Sign up at [Payman AI Dashboard](https://app.paymanai.com)
2. Switch to **Developer Mode** for testing with $1,000 in test funds
3. Navigate to **AI Agents → Default Agent**
4. Click **"Regenerate Credentials"** to get your API key

## 3. Installing Dependencies

Install [**Paykit**](/main/toolkit) and your preferred AI SDK:

```bash
# Install core packages
npm install payman-paykit ai

# Choose your AI provider
npm install @ai-sdk/openai   # For OpenAI
# or
npm install @ai-sdk/anthropic  # For Anthropic
```

<Note>
	PayKit is a unified Payman AI toolkit designed for AI agents, enabling smooth
	money handling through a single integration. It wraps all essential
	financial tool calls—payments, payees, and deposits—into one kit, working
	effortlessly with Vercel AI SDK, LangChain, and other LLM frameworks.
</Note>

## 4. Setting Up Paykit in Your API

Create a new API route: `app/api/chat/route.ts` 

```typescript
import { paykit } from "payman-paykit";
import { openai } from "@ai-sdk/openai";
import { streamText } from "ai";

// Initialize Paykit
const tools = paykit({
	apiSecret: process.env.PAYMAN_API_SECRET!,
	environment: "sandbox",
});

// Create chat endpoint
export async function POST(req: Request) {
	const { messages } = await req.json();
	const model = openai("gpt-4");

	const result = streamText({
		model,
		messages,
		tools,
		system: "You are a payment assistant that can send money and manage transactions.",
	});

	return result.toDataStreamResponse();
}
```

## 5. Configuring Environment Variables

Set up your `.env.local` file:

```bash
PAYMAN_API_SECRET=your_api_secret
PAYMAN_ENVIRONMENT=sandbox
OPENAI_API_KEY=your_openai_key  # If using OpenAI
```

<Warning>
	Ensure your environment variables are correctly set before running your app.
</Warning>

## 6. Implementing the Client-Side Chat Component

Create a new page component: `app/page.tsx`

```typescript
"use client";

import { useChat } from "ai/react";

export default function Chat() {
	const { messages, input, handleInputChange, handleSubmit } = useChat();

	return (
		<div>
			{messages.map((m) => (
				<div key={m.id}>
					{m.role}: {m.content}
				</div>
			))}

			<form onSubmit={handleSubmit}>
				<input
					value={input}
					onChange={handleInputChange}
					placeholder='Send a message...'
				/>
				<button type='submit'>Send</button>
			</form>
		</div>
	);
}
```

Key Features of `useChat` Hook:

-   Handles WebSocket connections automatically
-   Streams AI messages in real-time
-   Supports AI tool calls for payments
-   Maintains chat history effortlessly
