---

### Sign Up To Payman Platform

Go to the <a href="https://app.paymanai.com">Payman web app</a> and sign up for an account. Sign up as a developer and get onboarded.

### Create a Wallet

Allow your AI Agent access to the platform by creating a Wallet and getting your Payman API Key. This will give the Agent the ability to spend funds to create tasks to get work done.

<Frame caption="Create a Wallet on the Payman Platform">
  <img src="/images/add-agent.png" />
</Frame>

### Fund your Wallet & Get API Key

In order to create a new wallet you will need to add funds.  Currently we support USD and USDC.

<Frame caption="Fund your AI Agent Easily On Payman">
  <img src="/images/add-funds.png" />
</Frame>

Make sure you added a payment method for the currency you are trying to use as you will need it to fund your wallet.

<Note>
  You will also recieve your API Key here, be sure to store in a secure place
</Note>

### Start Creating Tasks With Funds

Now that your AI has access to funds, it can start creating tasks on Payman. Follow along with the documentation to learn how to assign tasks to specific users.
