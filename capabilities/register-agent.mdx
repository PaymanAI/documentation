title: Register AI Agent

---

### Sign Up To Payman Platform

Go to the <a href="https://app.paymanai.com">Payman web app</a> and sign up for an account. Sign up as a developer and get onboarded.

### Register Agent

Once you've registered your AI Agent you will recieve the API key which you can provide your agent for its custom access to the balances in the Agent Account.

<Frame caption="Register your Agent to your Account on the Payman Platform">
  <img src="/images/register-agent.png" />
</Frame>

<Note>
  In "Developer Mode" an AI Agent will automatically be registered for you.
</Note>

### Get API Key

In order to have your AI Agent get access to the capital save your API Key and integrate it into your AI Agent

<Frame caption="Fund your AI Agent Easily On Payman">
  <img src="/images/api-key-secret.png" />
</Frame>

Make sure you added a payment method you are trying to use as you will need it to fund your wallet.

<Note>
  You will need to have funds in your agent account for the agent to start
  moving money.
</Note>

### Start Making Payments

Now that your AI has access to funds, it can start paying for things. Follow along with the documentation to learn how to create payment destinations and
send funds.
