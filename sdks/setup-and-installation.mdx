<Tab title="Python">
### Installation
<CodeGroup>
```bash pip
pip install paymanai
```

```bash poetry
poetry add paymanai
```

```bash pipenv
pipenv install paymanai
```
</CodeGroup>

### Initialize Client
```python
from paymanai import Paymanai

payman = Paymanai(
    x_payman_api_secret="your_api_key",
    environment="sandbox"  # Use 'production' for live
)
```

### Parameters
| Parameter | Type | Description |
|-----------|------|-------------|
| `x_payman_api_secret` | str | Your API key from the [Payman Dashboard](https://app.paymanai.com/dashboard) |
| `environment` | str | `'sandbox'` or `'production'` |

### Initialize Client
```typescript
import Paymanai from "paymanai";

    xPaymanAPISecret: "your_api_key",
    environment: "sandbox"  // Use 'production' for live
});
```

### Parameters
| Parameter | Type | Description |
|-----------|------|-------------|
| `xPaymanAPISecret` | string | Your API key from the [Payman Dashboard](https://app.paymanai.com/dashboard) |
| `environment` | string | `'sandbox'` or `'production'` |
