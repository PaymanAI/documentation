<Tab title="Python">
### Installation
<CodeGroup>
```bash pip
pip install paymanai
```

```bash poetry
poetry add paymanai
```

```bash pipenv
pipenv install paymanai
```
</CodeGroup>

### Initialize Client
```python
from paymanai import Paymanai

payman = Paymanai(
    x_payman_api_secret="PAYMAN_ACCESS_KEY"
)
```

### Parameters
| Parameter | Type | Description |
|-----------|------|-------------|
| `x_payman_api_secret` | str | Your **Access Key** from the [Payman Dashboard](https://app.paymanai.com/dashboard) |


</Tab>

### Parameters
